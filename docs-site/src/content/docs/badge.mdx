Add a "Built with cckit" badge to your README — optional, never required, and a nod is always appreciated.

[![Built with cckit](https://img.shields.io/badge/built%20with-cckit-0A7E8C.svg)](https://github.com/jeiemgi/cckit)

## Markdown

```markdown
[![Built with cckit](https://img.shields.io/badge/built%20with-cckit-0A7E8C.svg)](https://github.com/jeiemgi/cckit)
```

## HTML

```html
<a href="https://github.com/jeiemgi/cckit"><img src="https://img.shields.io/badge/built%20with-cckit-0A7E8C.svg" alt="Built with cckit"></a>
```

## Flat-square variant

Append `?style=flat-square` to the badge URL for a flat look:

```text
https://img.shields.io/badge/built%20with-cckit-0A7E8C.svg?style=flat-square
```
