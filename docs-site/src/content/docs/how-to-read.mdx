This is a visual guide. Coloured **tags** tell you *how* a thing is used before you read a word of it.

When you see a tag next to a heading or a name, it tells you how that thing is triggered. The colour
is consistent everywhere — once you learn it here, you read it everywhere.

## The tags

<TagLegend />

## How to use them

- A page tagged <Tag kind="flow" /> describes a process you can run from start to finish.
- A name tagged <Tag kind="command" /> is something you type in a terminal.
- A name tagged <Tag kind="skill" /> is something you run inside Claude Code with a `/`.
- A name tagged <Tag kind="conversational" /> happens just by asking Claude in plain words.

You don't have to read top to bottom. Scan for the tag that matches what you want to do, and start
there.
